# Personal Resume / Theme

JSONResume.org theme(-ish) based of https://github.com/nass600/jsonresume-theme-elite

## Build steps

### Production

```sh
> npm start -- build
```

### Development

```sh
> npm start -- webpack.build.development
```

### Development Server

```sh
> npm start -- webpack.build.development.serve
```
